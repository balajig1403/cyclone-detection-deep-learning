# 🌪️ Cyclone Detection using Deep Learning

## 📌 Overview
This project focuses on detecting cyclones from satellite imagery using deep learning techniques. The model leverages temporal patterns in NetCDF satellite data to classify cyclone and non-cyclone events.

---

## 🚀 Features
- NetCDF data preprocessing
- Noise removal & normalization
- Temporal differencing (motion detection)
- CNN + LSTM model for sequence learning
- Class imbalance handling
- Model evaluation with AUC, Precision, Recall

---

## 🧠 Model Architecture
- TimeDistributed CNN layers
- LSTM for temporal learning
- Dense output layer (binary classification)

---

## 📊 Dataset
- Source: Satellite NetCDF files
- Classes:
  - Cyclone
  - Non-Cyclone

---

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- NetCDF4

---

## 📈 Results
- Achieved strong classification performance after preprocessing improvements
- Improved AUC and accuracy using temporal features
